A minimal personal finance tracker. Log income and expenses, see where your money goes, set budgets. No account, no cloud, works offline.

Features

💰 Income & expense logging
📅 Month-by-month navigation
📊 Daily flow chart
🗂️ Category breakdown with % share
🎯 Budget limits per category
🏷️ Custom categories with emoji
🌙 Dark & light mode
💾 Offline-first, data stays on your device
